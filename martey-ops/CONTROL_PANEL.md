# Martey Ops Control Panel

This is the “one page” launcher for running the martey-ops agent stack.

## Your Agent Stack
- Studio Chief (orchestration)
- Accessible Growth Strategist (Simply Accessible growth + copy)
- Digital Compliance Scout (ADA/WCAG/504 translator)
- Local Signal Hunter (local story leads)
- Field Research Operator (timelines + evidence tables) [optional if added]
- Family Voice Editor (publishable posts)
- Reality Gate (verification + risk)
- Knowledge Smith (templates + reuse)

---

# How to Run a Mission (Copy/Paste)

## Step 1 — Always start with Studio Chief
Copy/paste this into ChatGPT:

**ACTIVATE: Studio Chief**

Mission:
[write what you want done]

Context:
- Who it’s for:
- Location (if relevant):
- Links (if any):
- What you already know:

Constraints:
- Tone:
- Time window:
- Must include:
- Must avoid:

Deliverables:
- (list exactly what you want back)

## Step 2 — Then run the specialist outputs
After Studio Chief responds with assignments, copy/paste:

**ACTIVATE: [Agent Name]**
Task:
[paste the specific task Studio Chief assigned]

# Quality Gate (Always Use Before Posting)
Before publishing anything important, run:

ACTIVATE: Reality Gate

Task:
Review the draft for:
- Unsupported claims
- Missing context
- Risky wording
- What to verify next

Deliverables:
- Red flags
- Suggested safer rewrites
- Verification checklist
- Confidence rating

---

# Knowledge Capture (After a Good Result)
After you finish a mission, run:

ACTIVATE: Knowledge Smith

Task:

- a reusable template
- a checklist
- 3 improvements for next time
