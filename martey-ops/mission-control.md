# Martey Ops Mission Control

This file coordinates the martey-ops agent stack.

## Core Command

Studio Chief receives the mission first.

Studio Chief assigns tasks to:

- Accessible Growth Strategist
- Local Signal Hunter
- Family Voice Editor
- Reality Gate
- Knowledge Smith

## Standard Workflow

1. Mission description
2. Studio Chief planning
3. Research (Signal Hunter)
4. Writing (Family Voice Editor)
5. Verification (Reality Gate)
6. System capture (Knowledge Smith)

## Example Mission

"Identify a local issue in Lodi, gather facts, and produce a Facebook post for San Joaquin Family Voice."
