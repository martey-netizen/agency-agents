# Workflow 01 — Simply Accessible Leads

## Inputs
- Target audience: church OR school OR nonprofit
- City/region (optional)
- Website URL (optional)
- What you’re selling (audit / remediation / monthly care)

## Steps
1) Studio Chief — produce plan + assign agents
2) Accessible Growth Strategist — positioning + landing copy + outreach
3) Reality Gate — verify claims, tighten language, add caveats where needed

## Deliverables
- Landing page copy blocks
- 1 short outreach message + 1 longer version
- Objection-handling bullets
- Verification checklist
